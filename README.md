A professional-grade, AI-powered web application for detecting plant diseases from leaf images.

## Features
- **AI Detection**: Instant identification of plant diseases.
- **Scan History**: Automatically saves your past analyses.
- **Analytics Dashboard**: Visual charts of disease frequency.
- **PDF Reports**: downloadable professional reports.
- **PWA Support**: Installable as a native app on Desktop/Mobile.

## How to Run

1.  **Install Python Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Start the Server**:
    ```bash
    python main.py
    ```

3.  **Open the App**:
    - Go to [http://localhost:8000](http://localhost:8000) in your browser.

## Project Structure
- `main.py`: The FastAPI backend server.
- `model.py`: The simulated AI disease detection logic.
- `frontend/`: Contains all HTML, CSS, and JavaScript files.
- `requirements.txt`: List of Python libraries needed.
